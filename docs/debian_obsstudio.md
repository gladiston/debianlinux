# Instalação do OBS Studio via Flatpak

O **OBS Studio** (Open Broadcaster Software) é o aplicativo de código aberto padrão da indústria para **captura de tela** e **transmissão ao vivo (streaming)**. É reconhecido por sua robustez e flexibilidade no gerenciamento de múltiplas fontes de vídeo, áudio e transições de cena, sendo a escolha primária de profissionais e criadores de conteúdo.

Recomenda-se a instalação via **Flatpak** para garantir que o OBS Studio utilize as bibliotecas mais recentes e minimize problemas de dependência em distribuições como Debian e Ubuntu.

Execute o comando a seguir no terminal:

```bash
flatpak install flathub com.obsproject.Studio
```

Após a conclusão da instalação, o OBS Studio estará disponível e deve ser localizado no menu de aplicativos do seu ambiente gráfico de desktop.
