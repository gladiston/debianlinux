# DRAWONYOURSCREEN

**ADVERTÊNCIA:** Este programa é uma **Extensão do GNOME Shell** e foi projetado para funcionar perfeitamente no **Ambiente de Trabalho GNOME**. Seu funcionamento em outros ambientes (como KDE, XFCE ou Cinnamon) pode ser limitado ou exigir alternativas.

O **DrawOnYourScreen** é um aplicativo simples e útil projetado para permitir que o usuário **desenhe ou faça anotações diretamente sobre qualquer parte da tela** do seu desktop Linux. Ele é ideal para apresentações, tutoriais e videoconferências, pois permite destacar elementos, traçar setas ou fazer anotações rápidas sem interferir no conteúdo original das janelas.

-----

## Instalação no Linux (GNOME)

Existem dois métodos principais para instalar o DrawOnYourScreen em sistemas baseados em Debian/Ubuntu, sendo o segundo o mais confiável para versões recentes do GNOME.

### 1\. Instalação Padrão via APT

Este é o método mais simples, mas pode falhar ou resultar em uma versão incompatível se você estiver usando uma versão recente do GNOME Shell:

```bash
sudo apt install gnome-shell-extension-draw-on-your-screen
```

### 2\. Instalação Manual (Recomendada em Caso de Falha)

Se a instalação via APT falhar ou o aplicativo não funcionar devido à incompatibilidade de versão (um problema comum com extensões GNOME em sistemas em constante atualização), a solução mais confiável é seguir o método de instalação manual fornecido pelo desenvolvedor:

1.  **Visite a página oficial do projeto:**
    [https://codeberg.org/som/DrawOnYourScreen](https://codeberg.org/som/DrawOnYourScreen)

2.  **Siga as instruções de instalação** detalhadas fornecidas na página. Geralmente, o processo envolve baixar o código-fonte, movê-lo para a pasta de extensões do seu usuário (`~/.local/share/gnome-shell/extensions/`) e, em seguida, ativar a extensão.

Após a instalação (por qualquer um dos métodos), você precisará **ativar a extensão** usando o aplicativo "Extensões" ou a ferramenta "Ajustes" (GNOME Tweaks).
